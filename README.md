# react-boilerplate\

Uses Preact!

### Install
```sh
git clone https://github.com/davidli-minxiao/react-boilerplate.git
cd react-boilerplate
npm install
```

### Run the app from a dev server
```sh
npm serve
```

### Build the app
```sh
npm run build
```

### Host the app from a server (app must have been built)
```sh
npm run serve
```
